# BlueprintDB: Unified Database Management Tool

BlueprintDB is a professional, desktop-based database management application developed in C#. It is engineered for high performance, architectural clarity, and a streamlined experience for developers managing complex data environments.

## 🚀 Key Features

**Multi-Engine Support:** Seamlessly connect and manage 10 different database engines including:

- MS Access (full 64-bit support)
- SQL Server
- Firebird
- PostgreSQL, and more.

**Architectural Clarity:** Efficiently view and organize database metadata with a focus on structural visualization.

**Modern UI:** A clean, intuitive interface optimized for productivity and technical precision.

## 🛠 Tech Stack

- **Language:** C#
- **Platform:** .NET Windows Desktop
- **IDE:** Visual Studio / JetBrains Rider
